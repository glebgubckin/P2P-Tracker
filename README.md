## P2P Трекер v.1
Трекер связок P2P.

**Требования**:

 1. Node JS >= 16
 2. Yarn
 3. Запущенный сервер [P2P Server](https://github.com/OverTheGalaxy/P2P-Server)

**Установка**:

 1. git clone https://github.com/OverTheGalaxy/P2P-Tracker.git
 2. cd P2P-Tracker
 3. yarn
 4. yarn dev
 5. Перейти на http://localhost:5173
